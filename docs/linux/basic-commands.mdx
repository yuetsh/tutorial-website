## ls
列出所给目录下的所有文件和文件夹
```shell
ls / # 列出根目录
ls ~ # 列出当前用户的家目录
ls -al ~ # 列出当前用户家目录的详细信息
ll ~ # ls -l 的缩写
```

## cd
进入到某文件夹
```shell
cd / # 进入到根目录
cd ~ # 进入到当前用户的家目录
cd .. # 进入到上一级目录（退到上一级）
cd ../../ # 退到上上一级
cd ~/Desktop # 进入到家的 Desktop 目录
```
