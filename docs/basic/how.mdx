找到一门适合你的语言，然后就开始学习，推荐 C 语言、Python3 和 JavaScript 入门。

## 书籍或网站

菜鸟 [C 语言教程](https://www.runoob.com/cprogramming/c-tutorial.html) | [Python 教程](https://www.runoob.com/python3/python3-tutorial.html) | [JS 教程](https://www.runoob.com/js/js-tutorial.html)

- 免费，可运行，但是有些内容新手可能看不懂

廖雪峰 [Python 教程](https://www.liaoxuefeng.com/wiki/1016959663602400)

- 国内比较经典的 Python 教程

[C 语言中文网](http://c.biancheng.net/c/)

- 全面，但是有些部分要钱
