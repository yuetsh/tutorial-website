---
title: 英语不好怎么办？
sidebar_position: 4
---

虽然代码都是英文的，但是只要记住以下几个单词，就可以读懂代码。

|  单词   |     含义      |
| :-----: | :-----------: |
| include |     导入      |
|  stdio  | 标准输入输出  |
|  main   |     主要      |
|  print  |   打印/输出   |
| format  |    格式化     |
| return  |     返回      |
| decimal |    十进制     |
|  scan   |     扫描      |
|   if    |     如果      |
|  else   |     否则      |
|  while  | 当 ... 的时候 |
|   int   |     整数      |
|  float  |     小数      |
|  char   |     字符      |
