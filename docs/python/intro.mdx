## 第一次运行

恭喜你 🎉，你的第一个 Python 代码运行成功！

没错，Python 代码需要运行后，才可以执行出结果，在你点击运行按钮之后，Python 代码会被解释，这个原理我们之后会学到。

## 第一次编写

你可以把代码中**引号**里面的文字删掉，改成你的班级和名字，然后再次运行，看看会出现什么？

看来你不仅完成了第一次运行，也完成了 Python 语言的第一次编写。

当然运行成功的前提是我们要在**引号里面**修改文字，那么如果把引号不小心删掉呢？

## 第一次出错

没错，你可以试试把引号删掉一个，不用害怕，点击运行。

你会发现：

1. 文字颜色好像变了
2. 出现**运行失败**

:::danger 运行失败

SyntaxError: EOL while scanning string literal

:::

尤其是上面这句英文，你会觉得一头雾水，不用慌，我们把引号加回来

:::info

一定要是**英文**的引号，因为文字前面是**双引号**，所以我们也要使用**双引号**

:::

## 几个彩蛋

你一定注意到了我一直强调的**英文引号**和**双引号**，那么如果我们分别这样写：

1. 把双引号都改成单引号
2. 一个双引号，一个单引号
3. 英文双引号改成中文双引号

你可以都试试，总结一下哪些情况出现运行失败？

:::info

你一定会发现，在你点击运行按钮之前，如果文字高亮颜色变了，运行后就会出现错误。

这也算做一种错误提醒吧。

:::
