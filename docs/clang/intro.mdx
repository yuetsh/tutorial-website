C 语言是一门优秀的编程语言，是很多ita编程语言的基石，常年稳居 [语言榜](https://www.tiobe.com/tiobe-index/) 榜首或榜二。

最初 C 语言是用于系统开发工作，准确来说是就是编写操作系统（比如 Windows 操作系统）以及底层组件（比如硬件驱动、网络协议等）。因为 C 语言的优秀特性，一些其他的编程语言的编译器或者解释器就用 C 语言开发，所以 C 语言又被称为其他语言的“母语”。

具体来说，C 语言的应用有：

1. 操作系统

UNIX 是第一个使用高级语言设计的操作系统，它使用的编程语言就是 C 语言。后来，微软的 Windows、谷歌的 Android 也使用 C 语言编写。

2. 其他编程语言

世界上有上百种编程语言，其中比如 PHP、Lua、 Perl 等编程语言都是用 C 语言编写的，这是一门语言创造另一门语言的故事。

3. 嵌入式开发

C 语言是开发底层驱动的最佳选择，在物联网开发中，用的最多的就是 C 语言。当然什么是嵌入式开发，同学们还一头雾水，不过没关系，以后肯定会学习到。我们现在用遥控器控制空调、电视等，这里面肯定用到程序，这个程序就是嵌入式开发的一部分。

4. 游戏以及游戏引擎

有一款很著名的 FPS（第一人称视角射击）游戏《DOOM》，中文是《毁灭战士》，这款游戏完全是用 C 语言编写而成。

5. 数据库

用于保存数据、处理数据、查询数据的数据库大部分都是有 C 语言编写的，比如 MySQL、PostgreSQL 等。
:::note

为什么有些地方提到 C 语言的衍生语言 C++？

C 语言作为一门古老但又充满活力的语言，有很多新的特性加入到 C 语言中，这些加入的新特性形成了一门新的语言，叫做 C++ 语言，所以之后会用 C/C++ 代替。

:::
