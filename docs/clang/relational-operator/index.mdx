我们之前学过 C 语言中的加减乘除四则运算，分别是 + - \* / 这四个运算符，这里的运算符叫做「算术运算符」，因为它们负责**计算**

有了计算之后，如果我想比较两个数字的大小关系，那么我就需要使用到「**关系运算符**」

在 C 语言中有 **6 个** 关系运算符，分别是：

| 符号 |    意思    | 以下都是对的 |             值得注意的地方             |
| :--: | :--------: | :----------: | :------------------------------------: |
|  >   |    大于    |    3 > 2     |              不要写成 》               |
|  \<  |    小于    |    2 \< 3    |               不要写成《               |
|  ==  |    等于    |    3 == 3    |   **一个等号是赋值，两个等号是相等**   |
|  >=  | 大于或等于 |    3 >= 2    |              不要写成 =>               |
| \<=  | 小于或等于 |   2 \<= 3    |              不要写成 =\<              |
|  !=  |   不等于   |    3 != 2    | 不要写成中文感叹号（！=）也不要写成 =! |

## 注意点

### ⭐️ 赋值和等于的区别，初学者很容易错

下面是一段很简单的 C 代码，实现的功能是

> 申明整型变量 a，把 1 赋值给 a，把 a 的值输出出来。

```c showLineNumbers
#include<stdio.h>
int main()
{
    int a;
    // highlight-next-line
    a = 1;
    printf("a 的值是 %d", a);
    return 0;
}
```

第四行是一句 `赋值语句`，我们需要从右往左读：把 1 赋值给整型变量 a，所以一个等号是表示 `赋值`

:::tip

一个等号已经有含义了，所以在表示 `相等` 时候，需要**两个**等号。

:::

我们稍加改造，把上面的代码改成下面的样子。

```c showLineNumbers
#include<stdio.h>
int main()
{
    int a;
    a = 1; // 这里是赋值
    // highlight-next-line
    if (a == 1) { // 这里括号里面是等于
        printf("a 和 1 相等");
    }
    return 0;
}
```

这段代码的意思

> 申明整型变量 a，把 1 赋值给 a，（接下来是一个条件语句，不会的没事，我来翻译），如果 a 的值等于 1，就输出一句话：a 和 1 相等

`a == 1` 表示：变量 a 的值与 1 相等

:::tip

睁大眼睛好好看清楚！

:::

### < 和 > 跟我们之前写的 `#include<stdio.h>` 中 < > 的关系

`#include<stdio.h>` 这句话肯定是大家都会默写的，这里的 < > 表示的是尖括号

而在我们这里，< 表示 小于号，> 表示 大于号

关键看运算符两端是不是可以比较大小的东西，比如

`a > b` 肯定是表示 a 大于 b

### >= 表示大于「或」等于

举个栗子 🌰

如果 `a >= 10` 是对的，那么 a 的值可以是 10，也可以是 11，也可以是 12，以此类推，**重点是包括 10**

同理如果 `a <= 10` 是对的话，a 也包括 10 这个数。

### 运算符的左右可以有空格

这个是书上没有讲到的，在判断两个数字的大小关系时，你可以这样写 `a>10`，也可以 `a > 10`（注意 大于号 两端的空格）

「关系运算符」两端可以加个空格，这样你的代码看上去更优美。
