## 一个等号表示赋值（=）

```c showLineNumbers
#include<stdio.h>
int main()
{
    int a;
    // highlight-next-line
    a = 1;
    printf("a的值是%d", a);
    return 0;
}
```

第五行意思是 `把 1 赋值给 a`

## 两个等号表示等于（==）

表示两个数字相等

| 符号 | 意思 | 以下都是对的 |    值得注意的地方    |
| :--: | :--: | :----------: | :------------------: |
|  ==  | 等于 |   20 == 20   | **不要写成一个等号** |

:::warning 易错点

在比较相等的时候不要写成一个等号

:::
