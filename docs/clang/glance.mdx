## 第一段代码

下面是你遇到的第一段 C 语言代码，点击运行看看结果

<Editor>{`#include<stdio.h>\n
int main()
{
    printf("Welcome to 黄岩一职");
    return 0;
}`}</Editor>

## 仔细观察

1. 代码大部分都是英文，但是也会有中文
2. 有一些 **英文** 状态下的符号，从上到下分别是：

    | 井号 | 尖括号 | 小数点 | 小括号 | 花括号 | 双引号 | 分号 |
    |----|-----|-----|-----|-----|-----|----|
    | #  | <\> | .   | ()  | {}  | ""  | ;  |

3. 有些单词和符号有着 **不同的颜色**

## 解释结果

这段代码做了什么事呢？

不就是：

> 电脑屏幕上面出现 Welcome to 黄岩一职 这句话

👆 这是我们日常说的，但是毕竟我们正在学习 C 语言，这样说显得很不专业，我们要使用编程思维来转换一下：

> 在屏幕上**输出**一行**字符串** “Welcome to 黄岩一职”

:::info

1. “在电脑屏幕上显示文字” 叫做 **输出** 或者 **打印**

2. “单个文字” 叫做 **字符**，“多个字符连在一起” 叫做一个 **字符串**

3. 字符串需要用两个 **英文双引号** 包起来

:::

:::note 试一试

如果去掉双引号，会出现什么？

:::

## 解释代码

```c {5}
#include<stdio.h>

int main()
{
    printf("Welcome to 黄岩一职");
    return 0;
}
```

这段代码中，我们只需要关注高亮的这一行。

```c
printf("Welcome to 黄岩一职");
```

:::info

1. `printf` 是 `formatting the print` **格式化输出** 的缩写，对应上面的 **输出**

2. () 小括号里面是要输出的内容，显然这里输出的是一个字符串

3. 单独的 `Welcome to 黄岩一职` C 语言是不认得的，只有字符串 `"Welcome to 黄岩一职"` 才是有意义的

:::

当然还没有结束，接下来是 **很重要** 的一点。

中文的一句话结束使用句号。表示，在 C 语言中一行语句的结束使用 **分号 ;**

:::caution 注意

`printf("Welcome to 黄岩一职")` 没有分号会报错，不信你试试。

:::

所以

:::info

4. C 语言中一行完整的语句需要用 ; 结尾

:::
