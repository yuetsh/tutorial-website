## 开始挑战

坚持就是胜利，决赛圈开始了！在最后一圈，刘老师这样设计：

前三排一起喊：一班一班，实力非凡

后三排一起喊：众志成城，勇往直前

最后全班一起喊：一班最牛，最牛一班

已经到最后一关了，徐老师也来加亿点难度，这一关我不会给出思路，你们要靠自己，但是锦囊妙计还是有的。

## 完成挑战

<Editor language="c" showInput>{`#include<stdio.h>\n
int main()
{
    int n;
    scanf("%d", &n);
    // 当 n 不等于 4，那 n 就只能是 1 或者 2 或者 3，那就是表示前三排
    // 不等于怎么写呢，看看后面的锦囊吧
    if (n  4) {
        printf("一班一班\\n");
        printf("实力非凡\\n");
    }
    // 后三排，说明 n 取值是 2 或者 3 或者 4
    // 那 n 跟 1 是什么关系时，才能表示后三排
    if (n  1) {
        printf("众志成城\\n");
        printf("勇往直前\\n");
    }
    // 以下两句口号都是全班一起喊的，说明 n 可以是 1 或者 2 或者 3 或者 4
    // n 跟 4 是什么关系，能表示 n 是 1 或者 2 或者 3 或者 4 呢？
    if (n  4) {
        printf("一班最牛\\n");
    }
    // n 跟 1 是什么关系，能表示 n 是 1 或者 2 或者 3 或者 4 呢？
    if (n  1) {
        printf("最牛一班\\n");
    }
    return 0;
}`}</Editor>
